> 最后更新：2026-04-21

---

## 一、产品愿景

**一句话定位**：帮助美国房产投资者在5分钟内获取专业级投资分析报告

**核心价值**：将分散的房产数据（估值、租金、学区、犯罪率、地理因素）整合成一份易理解的投资决策报告，节省投资者数小时的调研时间。

---

## 二、目标用户画像

### 主要用户：个人房产投资者
- 年龄：30-55岁
- 收入：中产阶级，有闲钱投资
- 痛点：
  - 看中一套房，但不确定是否值得投资
  - 需要查多个网站（Zillow查价、学区网站查学校、警局网站查犯罪率）
  - 不懂如何计算投资回报率
- 愿意付费原因：节省时间，降低投资决策风险

### 次要用户：房产中介
- 需要快速生成报告给客户看
- 提升专业形象
- 愿意付费原因：提高成交效率

---

## 三、核心买点（用户为什么付费）

### 买点1：一站式数据整合（省时间）
| 传统方式 | PropertyPulse |
|---------|---------------|
| 查5个网站，花2小时 | 一个网站，5分钟 |
| 数据分散难以对比 | 一份报告全搞定 |

### 买点2：专业投资分析（降低风险）
提供投资者真正关心的指标：
- **现金流转正月数**：多久开始赚钱？
- **CAP Rate**：这个回报率值得投吗？
- **租售比**：租房还是买房更划算？
- **5年增值预测**：长期投资价值如何？

### 买点3：地理位置智能分析（独特优势）
利用地理科学专业知识，提供竞品没有的分析：
- **通勤便利度**：到主要就业中心距离
- **设施密度**：周边学校、医院、超市分布
- **灾害风险**：洪水、地震风险等级
- **社区趋势**：该区域人口增长、收入变化

### 买点4：可分享的专业报告
- PDF格式，可直接发给合作伙伴/贷款机构
- 包含数据来源说明，增加可信度
- 品牌定制（中介用户）

## 四、竞品分析

| 竞品 | 价格 | 优势 | 劣势 | 我们的突破点 |
|------|------|------|------|-------------|
| Zillow | 免费 | 数据全 | 无投资分析 | 添加投资回报计算 |
| Mashvisor | $49/月起 | 投资分析全 | 贵、无地理分析 | 更便宜+地理分析 |
| RentCast | API收费 | 数据准确 | 无界面 | 提供可视化界面 |
| NeighborhoodScout | $40/报告 | 学区/犯罪分析 | 按次付费贵 | 订阅制更划算 |

**差异化定位**：
> 比 Zillow 专业，比 Mashvisor 便宜，比单独买报告划算

## 六、技术架构

### 前端（Next.js）
```
frontend/
├── app/
│   ├── page.tsx           # 首页/着陆页
│   ├── dashboard/         # 用户仪表盘
│   ├── search/            # 房产搜索页
│   ├── report/            # 报告查看页
│   └── pricing/           # 定价页
├── components/
│   ├── PropertyCard.tsx   # 房产信息卡片
│   ├── AnalysisChart.tsx  # 分析图表
│   └── ReportPreview.tsx  # 报告预览
└── lib/
    ├── api.ts             # API调用
    └── utils.ts           # 工具函数
```

### 后端（FastAPI）
```
backend/
├── app/
│   ├── main.py            # 入口
│   ├── api/
│   │   ├── auth.py        # 认证接口
│   │   ├── properties.py  # 房产接口
│   │   └── reports.py     # 报告接口
│   ├── services/
│   │   ├── rentcast.py    # RentCast API
│   │   ├── geocoding.py   # 地理编码
│   │   └── report_gen.py  # 报告生成
│   ├── models/
│   │   ├── user.py
│   │   └── property.py
│   └── utils/
│       ├── investment.py  # 投资计算
│       └── geo_analysis.py # 地理分析
└── requirements.txt
```

### 数据库（PostgreSQL）
```
核心表：
- users          用户信息
- properties     房产数据缓存
- reports        生成的报告
- subscriptions  订阅记录
- search_history 搜索历史
```
